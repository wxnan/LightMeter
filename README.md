## 📱 应用简介

LightMeter 是一款专为Android设备设计的专业照度测量与计算工具。通过手机内置的光线传感器，实时测量环境光照强度，并提供专业的照度计算功能，帮助用户科学规划照明方案。

### 主要特点

- 📊 **实时测光** - 利用手机光线传感器，实时显示当前环境照度值
- 🧮 **照度计算** - 根据房间参数、灯具类型等因素，计算所需照度
- 🌿 **植物模式** - 内置50种常见植物的养护建议，提供光照和温度指导
- 🏠 **场景模式** - 涵盖多种生活场景的照明建议和照度标准
- 💾 **数据保存** - 支持保存测量记录，随时查看历史数据
- 🌓 **主题切换** - 支持深色/浅色主题切换，保护视力

## 🚀 使用指南

### 测光功能

1. 打开应用，默认进入测光页面
2. 授予应用必要的权限
3. 将手机放置在需要测量的位置
4. 实时查看照度值
5. 点击"开始记录"可连续记录数据
6. 点击"保存"将当前测量值保存到历史记录

### 照度计算

1. 切换到"计算"标签页
2. 选择房间类型
3. 输入房间长、宽、高
4. 选择期望照度或自定义照度值
5. 选择灯具类型和数量
6. 调整维护系数和利用系数
7. 点击"计算"查看结果

### 植物模式

1. 切换到"植物"标签页
2. 从列表中选择您的植物
3. 查看植物的光照和温度养护建议
4. 实时对比当前环境照度是否适合该植物

### 场景模式

1. 切换到"场景"标签页
2. 选择您的生活场景
3. 查看该场景的推荐照度范围
4. 获取专业的照明建议和注意事项
